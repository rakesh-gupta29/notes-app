A simple notes management app created using the MERN stack . 
This app creates notes in the form of a simple tiles based layout and supports full CRUD opertioan. 
Tech stack:
Mongo(Mongoose),
ExpressJS,
ReactJS(react-router-dom,react-context)

Platforms used:
Heroku:Backend ,
Netlify:Front-End,
Mongoose:Database
